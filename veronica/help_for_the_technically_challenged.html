  • Download the program + ciphertext by visiting here if you haven't already
  • Unzip the files to your Desktop
  • Open up Terminal (Just search for it)
  • Type "cd ~/Desktop" into Terminal --> cd stands for "change directory" and the tilda is your home directory. So basically you're telling the terminal to navigate to the Desktop.
  • Type "ls" --> ls stands for "list segments" and it lists all the files in your current directory. Somewhere in the listing should be two files: 'scrypto' and 'ITSYOBDAY.enc'
  • Type "cat ITSYOBDAY.enc" --> This prints out the contents of the file ITSYOBDAY.enc! This is what a ciphertext looks like - just seeming randomness
  • Type "cat scrypto | wc -l" --> This prints out how many lines the file scrypto is. THAT'S HOW MANY LINES OF CODE I WROTE FOR YOUR BDAY (jk it wasn't that many and I also had already made this)
  • Type "chmod +x scrypto" --> This makes sure you can execute the scrypto file (which is used to decrypt the file)
  • Type "./scrypto -d ITSYOBDAY.enc" and when prompted type in your real name as the password (AKA Veronica duhhhhh) --> This runs scrypto in decrypt mode on the file ITSYOBDAY.enc. If everything works correctly you should see "File: ITSYOBDAY.enc decrypted!"
  • Type "ls" --> There should be a new file called 'ITSYOBDAY.txt'
  • Type "cat ITSYOBDAY.txt" --> print your bday letter!
